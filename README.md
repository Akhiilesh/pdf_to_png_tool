# Cognex PDF to PNG Tool

A high-quality, fast, and secure PDF to PNG conversion tool. Everything happens right in your browser - no files ever leave your device.

![Cognex PDF to PNG Tool](docs-images/img.png)

## Features

- **High Quality**: Each PDF page is converted into a crisp PNG image.
- **Fast & Secure**: Processing is done locally on your machine.
- **User Friendly**: Simple drag-and-drop interface with a manual "Convert" button for better control.
- **Bulk Download**: Download individual pages or all of them in a convenient ZIP file.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [pnpm](https://pnpm.io/) or [npm](https://www.npmjs.com/)

### Installation

```bash
git clone [repository-url]
cd cognex-pdf-to-png
pnpm install
```

### Development

```bash
pnpm dev
```

### Build

```bash
pnpm build
```
